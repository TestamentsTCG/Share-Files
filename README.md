# UO Client Files

Drop all three `.mul` files into your Ultima Online Classic client folder:

```
C:\Program Files (x86)\Electronic Arts\Ultima Online Classic\
```

| File | Size | Description |
|------|------|-------------|
| `art.mul` | 140 MB | All tile and static art |
| `artidx.mul` | 1 MB | Art index |
| `tiledata.mul` | 3 MB | Tile properties |
